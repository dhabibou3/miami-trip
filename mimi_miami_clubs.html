Conseils Pratiques

Pour une soirée sans mauvaises surprises

💃 Ladies Night Rosa Sky Chaque mercredi, Rosa Sky organise un Ladies Night — entrée réduite ou gratuite. Parfait pour commencer la semaine en beauté avec vue sur le skyline.
👠 Dress code LIV LIV est très strict : talons obligatoires pour les femmes, pas de flats ni sandales. E11EVEN et LIV refusent shorts, sneakers et casquettes pour tout le monde.
📅 Réservez à l'avance LIV et E11EVEN se remplissent vite le week-end. Prenez vos tickets ou réservez une table minimum 48h avant pour éviter la longue file d'attente.
💸 Addition Dirty Rabbit The Dirty Rabbit ajoute automatiquement 20% de service charge. Vérifiez votre addition avant d'ajouter un pourboire supplémentaire !
🌙 E11EVEN après 1h du matin Le vrai spectacle commence après minuit. Seul club à continuer quand tous les autres ferment — parfait pour terminer la nuit en beauté.
🚗 Transport entre clubs Wynwood (Dirty Rabbit) → Brickell (Rosa Sky) → Downtown (E11EVEN) → South Beach (LIV). Prenez Uber/Lyft — ne conduisez pas entre les clubs la nuit.
